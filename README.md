## Project Overview
FixIQ is an AI-powered product support platform that helps users diagnose and fix product issues without reading lengthy manuals. Companies upload product documentation and users interact with an AI assistant that diagnoses issues like a real technician — asking follow-up questions to find the root cause.

## Features and Functionality
- Product Marketplace — Browse and search products by category
- AI Diagnostic Assistant — Asks smart follow-up questions to diagnose issues
- Knowledge Repository — PDFs and manuals uploaded per product
- Company Portal — Companies can add and manage their products

## Tech Stack Used
- React.js — Frontend UI
- Groq AI API (LLaMA 3.3 70B) — AI diagnostic assistant
- Netlify — Live deployment
- GitHub — Version control

## Setup and Installation Instructions
1. Clone the repository: `git clone https://github.com/aksshit-rathi/Mantis`
2. Navigate to project: `cd Mantis`
3. Install dependencies: `npm install`
4. Add your Groq API key in `src/App.js` line 13
5. Run locally: `npm start`
6. Open browser at `http://localhost:3000`

## Usage Guide
1. Open the live link below
2. Browse products on the marketplace
3. Click any product to open its page
4. Type your issue in the chat assistant
5. Answer the AI's follow-up questions
6. Get a precise diagnosis and fix
