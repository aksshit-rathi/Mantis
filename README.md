## Problem
Finding help for broken products is frustrating. Manuals are 100+ pages, information is scattered, and users give up and call technicians for simple fixes.

## Solution
FixIQ lets companies upload product manuals. Users chat with an AI assistant that diagnoses issues like a real technician — asking follow-up questions to find the root cause.

## Features
- Product Marketplace — Browse and search products
- AI Diagnostic Assistant — Asks smart questions to diagnose issues
- Knowledge Repository — PDFs and manuals per product
- Company Portal — Companies add and manage products

## Tech Stack
- React.js — Frontend
- Groq AI (llama-3.3-70b) — AI Assistant
- Netlify — Deployment
- GitHub — Version Control

## Setup Instructions
1. Clone the repo
2. Run `npm install`
3. Add your Groq API key in `src/App.js`
4. Run `npm start`
